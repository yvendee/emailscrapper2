# emailscrapper2


## Assume you have already install python 3 ,To run the script

Python script named main.py located in a folder named emailscrapper2 on the desktop.

### macOS:

```
cd ~/Desktop/emailscrapper2
```
```
python3 main.py
```


### Windows:
```
cd Desktop\emailscrapper2
```
```
python main.py
```
